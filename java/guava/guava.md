# Guava

guava的优点：

- **高效设计良好的API，被Google的开发者设计，实现和使用**
- **遵循高效的java语法实践**
- **使代码更刻度，简洁，简单**
- **节约时间，资源，提高生产力**

Guava工程包含了若干被Google的 Java项目广泛依赖 的核心库，例如：

- 工具栏 [Joiner/Spliter]
- 集合 [collections]
- 缓存 [caching]
- 原生类型支持 [primitives support]
- 并发库 [concurrency libraries]
- 通用注解 [common annotations]
- 字符串处理 [string processing]
- Cache 处理
- EventBus处理
- I/O 等等。
